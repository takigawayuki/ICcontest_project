# 车牌数据集分类报告

> 参考标准：《中华人民共和国机动车号牌》GA36-2018
> 比赛要求：常见车牌（蓝牌 + 绿牌）、特殊车牌（大黄牌、小黄牌、警用、挂车、领馆、使馆）

---

## 一、车牌分类定义（依据题目要求）

### 1.1 常见车牌

| 车牌类型 | 俗称 | 号牌种类 | 底色 | 文字颜色 | 格式 | 说明 |
|----------|------|----------|------|----------|------|------|
| 小型汽车号牌 | 蓝牌 | 小型汽车 | 蓝底 | 白字 | 省份缩写 + 字母 + 5位字符（共7位） | 最常见，普通私家车 |
| 小型新能源车号牌 | 绿牌 | 新能源小型汽车 | 渐变绿底 | 黑字 | 省份缩写 + 字母 + 6位字符（共8位） | 纯电/插混小型车 |

### 1.2 特殊车牌

| 车牌类型 | 俗称 | 底色 | 文字颜色 | 格式 | 说明 |
|----------|------|------|----------|------|------|
| 大型汽车号牌 | 大黄牌 | 黄底 | 黑字 | 省份缩写 + 字母 + 5位字符 | 货车、大巴等大型车辆 |
| 摩托车号牌 | 小黄牌 | 黄底 | 黑字 | 省份缩写 + 字母 + 4位字符 | 摩托车，尺寸较小 |
| 警用号牌 | 警牌 | 白底 | 黑字（含"警"字） | 地区代码 + 数字 + "警" | 公安警用车辆 |
| 挂车号牌 | 挂车牌 | 黄底 | 黑字（含"挂"字） | 省份缩写 + 字母 + 5位字符 + "挂" | 半挂车、全挂车 |
| 领事馆号牌 | 领馆牌 | 黑底 | 白字（含"领"字） | 数字 + "领" + 数字 | 外国领事馆车辆 |
| 大使馆号牌 | 使馆牌 | 黑底 | 白字（含"使"字） | 数字 + "使" + 数字 | 外国大使馆车辆 |

**标注格式说明：**
CCPD 系列均采用文件名内嵌标注，无需额外标注文件，文件名格式如下：

```
00360785590278-91_265-311&485_406&524-406&524_313&520_311&485_402&489-0_0_3_24_28_24_31_33-117-16.jpg
│              │       │                  │                              │                   │   │
│              │       └ BBox(x1,y1,x2,y2)└ 四顶点坐标                   └ 车牌字符索引         │   └亮度
└时间戳         └拍摄角度                                                                      └模糊度
```

车牌字符索引解码规则：
- **位置 0**（省份）：PROVINCES 表，共 34 项（皖=0, 沪=1, 津=2...）
- **位置 1**（字母位）：ALPHABETS 表，25 项（A-Z 去除 I，含 O）
- **位置 2 起**（字母数字）：ADS 表，35 项（A-Z 去除 I，0-9，含 O）
- CCPD2019 蓝牌：7 个索引（省份+1字母+5字符）
- CCPD2020 绿牌：8 个索引（省份+1字母+6字符，含 D 或 F 表示新能源）

## 四、BDD100K 的正确用途

BDD100K 是美国伯克利大学的驾驶场景数据集，**不含中国车牌**，但包含以下可用标注：

| 标注类别 | 数量（train） | 比赛用途 |
|----------|---------------|----------|
| person（行人） | 91,435 条 | 行人检测模型训练 |
| car（车辆） | 714,121 条 | 车辆检测辅助 |
| traffic light（红绿灯） | 186,301 条 | 闯红灯判断辅助 |
| bus / truck | 41,700 条 | 车辆类型识别 |

**推荐用途**：用于训练行人检测模型，结合红绿灯状态实现"行人闯红灯"检测，对应**创新加分项（5分）**。
