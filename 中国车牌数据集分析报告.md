## 二、CCPD2019 详细子集说明

CCPD2019 全部为**蓝牌**（标准中国小型车号牌），按拍摄条件分为以下子集：

| 子集 | 数量 | 拍摄条件 | 对比赛的价值 |
|------|------|---------|------------|
| `ccpd_base` | 199,996 张 | 标准场景，不同角度/距离/光照 | ✅ 蓝牌训练主力 |
| `ccpd_tilt` | 30,216 张 | 水平+垂直双向倾斜 | ✅ 覆盖"角度不限"要求 |
| `ccpd_challenge` | 50,003 张 | 最复杂的综合难度场景 | ✅ 提升鲁棒性 |
| `ccpd_blur` | 20,611 张 | 图像模糊 | ✅ 提升抗模糊能力 |
| `ccpd_fn` | 20,967 张 | 远距离或极近距离拍摄 | ✅ 提升多尺度适应 |
| `ccpd_rotate` | 10,053 张 | 水平方向大角度旋转 | ✅ 覆盖"角度不限"要求 |
| `ccpd_db` | 10,132 张 | 暗光或强曝光 | ✅ 提升光照适应 |
| `ccpd_weather` | 9,999 张 | 雨雾天气 | ✅ 提升天气适应 |
| `ccpd_np` | 3,036 张 | 无车牌（负样本）| ✅ 降低误检率 |

> **注意：** `ccpd_np` 文件名格式不同（如 `1005.jpg`），无标注信息，作为负样本使用。

## 三、CCPD2019 预定义数据集划分

`splits/` 目录已提供针对 `ccpd_base` 的官方划分：

| 文件 | 数量 | 用途 |
|------|------|------|
| `train.txt` | 100,000 张 | 训练集 |
| `val.txt` | 99,996 张 | 验证集 |
| `test.txt` | 141,982 张 | 测试集（含部分复杂场景）|

其余子集（blur/challenge/db 等）作为**独立测试集**使用，用于评估模型在特定场景下的表现。

## 五、标注格式说明

CCPD2019 与 CCPD2020 标注信息均编码在**文件名**中，格式一致：

```
025-95_113-154&383_386&473-386&473_177&454_154&383_363&402-0_0_22_27_27_33_16-37-15.jpg
 ①    ②         ③                  ④                       ⑤                 ⑥  ⑦
```

| 字段 | 内容 |
|------|------|
| ① | 车牌面积占图片比例 |
| ② | 水平倾斜度_垂直倾斜度 |
| ③ | BBox：左上角&右下角坐标 |
| ④ | 四个角点坐标（从右下角起，顺时针）|
| ⑤ | 车牌字符编码（省份+字母+5位字符）|
| ⑥ | 亮度 |
| ⑦ | 模糊度 |

## 六、与比赛要求的对应关系

比赛要求："角度、尺寸、数量等参数皆不做标准限定"

| 比赛场景 | 对应数据集 |
|---------|-----------|
| 正常拍摄 | ccpd_base |
| 倾斜车牌 | ccpd_tilt + ccpd_rotate |
| 模糊图像 | ccpd_blur |
| 远近距离 | ccpd_fn |
| 暗光/强光 | ccpd_db |
| 雨雾天气 | ccpd_weather |
| 复杂综合 | ccpd_challenge |
| 无车牌场景 | ccpd_np（负样本）|
| 绿牌（新能源）| CCPD2020 ccpd_green |
